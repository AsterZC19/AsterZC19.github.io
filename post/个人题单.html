# CodeForces|1624D. Palindromes Coloring

  • 题意
    • 给定一个长度为 nnn 的字符串，可以给其中一些字符染色（可以不染），颜色种类不超过 kkk。然后将同种颜色的字符进行任意交换。
    • 定义一个颜色串为，所有染上此颜色的字符按照 原串顺序（可以为交换后） 组成的字符串。
    • 你要使得所有颜色串均为回文串，并且所有颜色串长度的最小值最大。求这个最大值。
    • 多组测试
  • 题解
    • 由要求的 最小值最大 可知，该题可以使用二分，下证：
    • 假设现在颜色串长度的最小值为 lenlenlen ，那么所有长度 >len> len>len 的颜色串一定可以通过删除某些字符（相当于不染一些字符），变成一个长度为 lenlenlen 的回文串。
    • 也就是说，我们只需判断是否存在 kkk 个长度为 lenlenlen 的回文颜色串即可。
    • 同理，如果颜色串长度为 lenlenlen 可以，那么 len−1len - 1len−1 也可以。这就说明了 lenlenlen 具有单调性，故可以使用二分。
    • 又发现我们可以不限次数地交换同颜色字符。这说明我们可以将颜色串字符的顺序进行任意变换，得到我们想要的颜色串。
    • 所以，我们只需知道同颜色的字符有哪些，就可以快速判断它是否为回文串。统计出每个字符出现的次数，如果它是回文串，要么出现的次数均为偶数（颜色串长度为偶数），要么只有一个奇数（颜色串长度为奇数）。
    • 据此，我们可以使用一个桶记录下整个字符串中各个字符出现的次数。
      • 若 lenlenlen 为偶数，我们就要为 kkk 种不同颜色串提供 len2\frac{len}{2}2len​ 对相同字符，总共就是 k×len2k \times \frac{len}{2}k×2len​ 对。直接判断 $\sum_i = 1}({26)\left \lfloor \frac {cnt_i}{2} \right \rfloor $ 是否大于等于 k×len2k \times \frac{len}{2}k×2len​ 即可。
      • 若 lenlenlen 为奇数，同理，先判断 $\sum_i = 1}({26)\left \lfloor \frac {cnt_i}{2} \right \rfloor $ 是否大于等于 k×⌊len2⌋k \times \left \lfloor \frac{len}{2} \right \rfloork×⌊2len​⌋，因为长度为奇数的回文串还有中间一个落单的字符，所以还需判断剩下未使用的字符是否大于等于 kkk。
    • 时间复杂度 O(26nlogn)O(26n log n)O(26nlogn)

# CodeForces|1610C. Keshi Is Throwing a Party

  • 题意
    • 有 nnn 个人，第 iii 个人有 iii 元。现在要邀请若干个人参加聚会若
      • 钱数大于 iii 的人的个数小于等于 aia_iai​。
      • 钱数小于 iii 的人的个数小于等于 bib_ibi​。
    • 则这个人会高兴，求 最多 邀请多少个人使得每个参加宴会的人都会高兴。
  • 题解
    • 题意中出现了 最多，故考虑二分。
    • 假设存在邀请 kkk 个人的方案，那么邀请 k−1,k−2...1k - 1, k - 2 ... 1k−1,k−2...1 个人都一定可以。反过来如果 kkk 个人的方案不行，那么 k+1,k+2...nk + 1, k + 2 ... nk+1,k+2...n 个人的方案一定都不行。
    • 因此该题可以对 能够邀请的人数 进行二分。
    • 之后考虑贪心判断人数是否可行。
    • 现在的目标是根据已选的总人数判断我们可以选出的最多人数。
    • 对于第 iii 个人，如果 bi≥cntb_i \ge cntbi​≥cnt 并且 ai≤sum−cnt−1a_i \le sum - cnt - 1ai​≤sum−cnt−1，此时如果邀请 iii ，他会很开心，所以我们一定会邀请他。因此扫一遍所有人，我们就得到了实际选择的人数，最后将其与 sumsumsum 比较即可。

# CodeForces|1613D. MEX Sequences

  • 题意：求满足以下条件的子序列 (a1,...,an)(a_1, ... , a_n)(a1​,...,an​) 的数量。

    • 对任意的 aia_iai​ ，有 ∣ai−MEX(a1,a2,...,ai)∣≤1| a_i - MEX(a_1, a_2, ... , a_i)| \le 1∣ai​−MEX(a1​,a2​,...,ai​)∣≤1 。

  • 题解

    • 我们考虑子序列如何拓展。
    • 设 1.1.1. 类子序列满足以下条件：最大值为 xxx ，且包含了 [0,x][0, x][0,x] 中的所有整数。则其下一个数可选 x,x+1,x+2x, x + 1, x + 2x,x+1,x+2 。如果选前两个，则其仍然是 1.1.1. 类子序列。如果选择 x+2x + 2x+2 ，则转化为另一类子序列。
      • 空序列是 1.1.1. 类子序列。
    • 设 2.2.2. 类子序列满足以下条件：最大值为 xxx ，且包含了 [0,x−2][0, x - 2][0,x−2] 的所有整数，却不包含 x−1x - 1x−1 。则其下一个数可选 x−2x - 2x−2 或 xxx 。无论选择哪一个，其 MEXMEXMEX 值均不改变。
    • 所以，所有满足条件的序列都可以分为这两类。
    • 设 dp[x][0/1]dp[x][0/1]dp[x][0/1] 是最大值为 xxx 的 1./2.1. / 2.1./2. 类子序列的数量，则可以进行状态转移。

# CodeForces|1638C. Inversion Graph

  • 题意
    • 给定排序 ppp，对于逆序对 (pi,pj)(p_i, p_j)(pi​,pj​)，连接一条 i⟷ji\longleftrightarrow ji⟷j 的双向边。求整个图的连通块数量。
    • 多组测试。
  • 题解
    • 考虑单调栈，栈内的每一个元素代表一个独立的连通块（中的元素最大值）。
    • 维护方法如下
      • 枚举 i=1∼ni = 1 \sim ni=1∼n 入栈。
      • 如果 aia_iai​ 小于栈内的某些元素 t1,t2,...,txt_1, t_2, ... , t_xt1​,t2​,...,tx​ ，说明这些元素包括 aia_iai​ 同属于一个连通块，故弹出这些元素，以该连通块中的一个最大元素代表这个连通块。
      • 如果当前 aia_iai​ 大于栈内所有的元素，则认为其与前面的连通块之间没有任何直接连边，可认为其为当前状态下是一个最大元素为 aia_iai​ 的新的连通块。
    • 时间复杂度为 O(n)O(n)O(n)。

# CodeForces|1562B. Scenes From a Memory

  • 题意

    • 给定一个 kkk 位数，要求对其删除若干位后使其是一个 非质数 （等于 111 或是一个合数），该数中不含 000。
    • 多组测试。保证有解。

  • 题解

    • 引理 对于一个三位数 nnn ，你总可以找到一种方案，删掉至少 111 位，使其剩下最多 222 位且是一个 非质数。
    • 证明 注意到可以分成以下几种情况考虑：
      1. nnn 中本身含有 1,4,6,8,91, 4, 6, 8, 91,4,6,8,9 之一，只保留这一位即可。
      2. nnn 中第 i(i>1)i(i > 1)i(i>1) 位是 2,52, 52,5 之一，保留第 1,i1, i1,i 位即可。
      3. nnn 中有两个相同的数，只保留这两个相同的数即可。
      4. nnn 中第 111 位是 2,52, 52,5 之一，第 i(i>1)i(i > 1)i(i>1) 位是 777，保留第 1,i1, i1,i 位即可。
    • 运用数学归纳法可以把该 引理 拓展至 kkk 位数。
    • 同时，根据数据范围来看，只需要特判个位数，两位数只需要遍历一遍，判断是否为合数即可，故在此预处理 100100100 以内的所有数。
