# 树与图的存储

  • 树是一种特殊的图，与图的存储方式相同。
  • 对于无向图中的边 ab，存储两条有向边 a->b, b->a。

# 拓扑排序

  • 时间复杂度 O(n+m)O(n+m)O(n+m)，n 表示点数，m 表示边数

# 朴素 dijkstra 算法

  • 时间复杂度是 O(n2+m)O (n^2 + m)O(n2+m)，n 表示点数，m 表示边数

# 堆优化版 dijkstra

  • 时间复杂度 O(mlogn)O (mlogn)O(mlogn)，n 表示点数，m 表示边数

# Bellman-Ford 算法

时间复杂度 O(nm)O(nm)O(nm), n 表示点数，m 表示边数
注意在模板题中需要对下面的模板稍作修改，加上备份数组，详情见模板题。

# spfa 算法（队列优化的 Bellman-Ford 算法）

  • 时间复杂度平均情况下 O(m)O (m)O(m)，最坏情况下 O(nm)O (nm)O(nm)，n 表示点数，m 表示边数

# spfa 判断图中是否存在负环

  • 时间复杂度是 O(nm)O (nm)O(nm)，n 表示点数，m 表示边数

# floyd 算法

  • 时间复杂度是 O(n3)O (n^3)O(n3)，n 表示点数

# 朴素版 prim 算法

  • 时间复杂度是 O(n2+m)O(n^2+m)O(n2+m)，n 表示点数，m 表示边数

# Kruskal 算法

  • 时间复杂度是 O(mlogm)O(mlogm)O(mlogm)，n 表示点数，m 表示边数

# 染色法判别二分图

  • 时间复杂度是 O(n+m)O(n+m)O(n+m)，n 表示点数，m 表示边数

# 匈牙利算法

  • 时间复杂度是 O(nm)O(nm)O(nm)，n 表示点数，m 表示边数
