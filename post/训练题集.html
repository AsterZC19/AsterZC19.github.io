# 第二周

# CF451E | Devu and Flowers

  • 题意
    • Devu 有 nnn 个花瓶，第 iii 个花瓶里有 fif_ifi​ 朵花。他现在要选择 sss 朵花。
    • 你需要求出有多少种方案。两种方案不同当且仅当两种方案中至少有一个花瓶选择花的数量不同。
    • 答案对 109+710^9 + 7109+7 取模。
  • 分析
    • 假如没有花瓶中对花数量的限制，我们可以用隔板法分隔出来不同的方案，我们要在 sss 朵花中插入 n−1n - 1n−1 块板，第一块板有 (s+1)(s + 1)(s+1) 种选择，最后的总数是 (s+1)×(s+2)×⋯×(s+n−1)=(s+n−1)!s!(s + 1) \times (s + 2) \times \cdots \times (s + n - 1) = \frac{(s + n - 1)!}{s!}(s+1)×(s+2)×⋯×(s+n−1)=s!(s+n−1)!​ 。
    • 但是方案数量与板的顺序无关，所以我们还要去掉这些板全排列的数量，即 Cs+n−1n−1C^{n - 1}_{s + n - 1}Cs+n−1n−1​ 。
    • 现在加入条件，我们只要把不符合条件的方案减去即可。
    • 首先我们要从所有的花中取出 fi+1f_i + 1fi​+1 朵在花瓶 iii 中的花，这样无论如何都是不符合条件的，方案数是 Cs+n−1−(fi+1)n−1C^{n - 1}_{s + n - 1 - (f_i + 1)}Cs+n−1−(fi​+1)n−1​。
    • 之后就考虑使用容斥原理，因为我们之前减掉的方案有重复的，又因为有 1≤n≤201 \le n \le 201≤n≤20，我们可以用状态压缩枚举每种情况，又因为组合数范围很大，我们直接使用逆元求组合数。

# SP32952 | ADAFTBLL - Ada and Football

  • 题意
    • 你有一棵共 nnn 个节点的树，一共有 qqq 次操作，操作有两种分别为：
      1. 1 x y ：表示把第 xxx 个节点的权值变成 yyy。
      2. 2 u v ：输出 ∑s×(s−1)2\sum \frac{s \times (s - 1)}{2}∑2s×(s−1)​，其中 sss 为 uuu 到 vvv 节点中出现的不同权值的次数。
  • 分析
    • 树上带修莫队。
    • 树上莫队的宗旨是把树变成序列，就能按普通序列处理了，这里要用到两个知识点 ：重链剖分 + 欧拉序。
    • 对于 a−ba - ba−b 的路径，分两种情况讨论。
      • aaa 和 bbb 在一条链上，途中的节点都只在 aaa 和 bbb 第一次出现的 dfs 序之间。
      • aaa 和 bbb 不在一条链上，那途中节点则为 aaa 最后一次出现的地方到 bbb 第一次出现的地方之间只出现一次的节点，aaa 最后一次出现因为把 aaa 的子树遍历完了， 之所以只出现一次和上面一样，不过最后还要加上它们的 lcalcalca，因为 a 和 b 都在 lca(a,b)lca(a, b)lca(a,b) 的子树里，lcalcalca 没有遍历到。
    • 注意，需要开一个数组记录节点的访问次数，如果访问一次就 del，没有就 add，然后取反即可。
