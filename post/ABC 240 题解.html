# C - Jumping Takahashi

  • 题目链接
  • 题意
    • 给出 NNN 组数 ai,bia_i, b_iai​,bi​。从数轴的 000 处开始每次可以选择两者之一的距离正向前进，问 NNN 步后能否到达 XXX 点。
  • 题解
    • 假设直接枚举的话有 2N2^N2N 种状态，肯定会超时。
    • 可以注意到
      • 如果多种走法可以到达一个点，则这些走法可以看作是一样的。
      • 一旦当前的坐标超过了 XXX，则他不需要再考虑了。
    • 因此我们考虑 DPDPDP
    • 状态表示
      • dp[i][j]=1/0dp[i][j] = 1/0dp[i][j]=1/0 表示能否在第 iii 步到达 jjj。
    • 状态计算
      • 假设我们已经计算到 kkk 步。
      • 对所有的 jjj，将 dp[k+1][j]dp[k + 1][j]dp[k+1][j] 初始化为 000。
      • 对所有的上一步中 dp[k][j]=1dp[k][j] = 1dp[k][j]=1 的 jjj。
        • 如果 j+ak≤Xj + a_k \le Xj+ak​≤X，则 dp[k+1][j+ak]=1dp[k + 1][j + a_k] = 1dp[k+1][j+ak​]=1。
        • 如果 j+bk≤Xj + b_k \le Xj+bk​≤X，则 dp[k+1][j+bk]=1dp[k + 1][j + b_k] = 1dp[k+1][j+bk​]=1。
    • 最后只需要判断 dp[N][X]dp[N][X]dp[N][X] 是否为 111 即可。
  • 优化
    • 然而我们可以注意到，每一步都是由上一步转移过来的，因此我们可以考虑状态压缩进行优化，用一个数维护所有的状态，并动态记录。

# D - Strange Balls

  • 题目链接
  • 题意
    • 给定 NNN 个权值大于 222 的小球，要将他们一个个塞入圆柱体，如果恰有 kkk 个权值为 kkk 的小球，则他们会消失。
    • 求每次塞入球后，圆柱体中有多少球。
